Principles and handoff

Eight rules that make v2 feel like one hand drew it, then how to read this file. If a decision is not covered by a specimen, decide with these.

01

One coral per surface. A screen earns exactly one solid coral action; everything else steps down to filled grey, bordered white, or ghost.

02

Warm neutrals, never cool slate. Every grey leans plum or putty; if a hex reads blue, it is from the old system.

#F2F1EF warm #F1F5F9 cool
03

Depth is a white card on a warm canvas. Shadows are micro and warm-tinted, reserved for active and raised controls, never for containers.

Active card --sh-nav
04

Negative tracking scales with size. Roughly -1% at 14 medium, -2.5% at 13 semibold and above at display sizes. Big type tightens, small type barely.

Headline meta at 13 -.025em / -.01em
05

Statuses are squares, plans are pills. Radius 4 tinted chips for content state; the bordered blue pill is reserved for billing.

Published Draft Team plan
06

Secondaries are filled, not outlined. The grey fill is the workhorse button; borders belong to tertiary actions like pagination and Preview.

07

Sidebar owns workspace nav, tabs own project nav. The burgundy bar marks where you live; the coral underline marks what you are editing.

PagesContentSchema
08

Predicted values need design sign off. Hover and pressed steps, the dark surface hexes, both neutral ramps, every overlay, and the light editor are extensions, not extractions.

How to use this file

Reading guide
Coral chips like h-40 · r-6 are measured straight from Figma dev mode. Treat them as law.
Amber chips like predicted mark values the Figma file does not specify yet. Build with them, but flag them in design review before they ship.
Use the theme toggle in the top right corner. Every specimen except the pinned proofs re-renders from the same variables, which is the point.
Everything interactive here is real: buttons hover and press, menus open, dialogs trap Escape, toasts dismiss themselves. If a state is frozen, its label says so.
Questions about a value: check the specimen chip first, then the Figma section 58408:8719, then ask design.
